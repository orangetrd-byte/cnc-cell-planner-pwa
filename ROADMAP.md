# CNC Cell Planner PWA Roadmap

## Phase 1: Shop-Flow Stability

- Keep travelers, work orders, operations, and move quantities reliable.
- Preserve offline-first PWA behavior.
- Keep assistant context tied to production planning, not machining reference tools.
- Maintain build information, version number, and MGP identity.

## Phase 2: Dispatch and Reports

- Improve machine queues and dispatch visibility.
- Strengthen completed-operation reporting.
- Add clearer bottleneck and late-work indicators.
- Improve import/export and recovery behavior.

## Phase 3: Scheduling Engine

- Add dependable capacity and due-date logic.
- Forecast completion from operation routing and machine availability.
- Keep scheduling rules testable before adding advanced charts.

## Phase 4: Production Polish

- Refine mobile and desktop layouts for repeated shop use.
- Add better empty states, validation, and print/handoff outputs.
- Keep the PWA focused on production flow.
