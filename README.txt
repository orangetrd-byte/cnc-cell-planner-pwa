CNC Cell Planner PWA

To run as an installable app during local testing:

1. Right-click start-app.ps1 and choose "Run with PowerShell".
2. Open http://localhost:8787 if the browser does not open automatically.
3. Use the browser install/app button to install the PWA.

Do not open index.html directly with file:// if you want PWA behavior.
Service workers require localhost or HTTPS.

Your job data and API keys are stored only in the browser's local storage.
